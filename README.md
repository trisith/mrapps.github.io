# MrApps - Professional Freelance Developer

A Jekyll-based portfolio landing page for GitHub Pages showcasing freelance development services.

## Features

- Modern, responsive design
- Smooth scrolling navigation
- Services showcase
- Portfolio section
- Skills display with progress bars
- Contact form

## Deployment

Push to `main` branch - GitHub Pages builds automatically.

## Local Development

```bash
bundle install
bundle exec jekyll serve
```

## Customization

Edit `index.html` to update:
- **Contact email** - Replace `your-email@example.com`
- **Form endpoint** - Replace `formspree.io/f/your-form-id` with your Formspree ID
- **Portfolio projects** - Update project cards
- **Skills** - Modify skill names and percentages
- **Social links** - Update GitHub, LinkedIn, Twitter URLs in footer
